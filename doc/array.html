Array


Derived from:Root

Description:
Arrays are fixed-size arrays, which can be manually expanded via the setSize method. Changing the size via setSize does not invalidate the original contents still in range. You can store and retrieve elements into/from an array. There does not exist an array literal as in Smalltalk.
