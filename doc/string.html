String


Derived from:Root

Description:
String instances represent character strings. String literals, such as "hello, world," are compiled into String instances, which can then be manipulated like other objects (i.e., you can send them messages).

A string can be converted to Integer and Float objects, as is done in C/C++ with atoi and atof. You can extract substrings and manipulate individual characters in a string. You can also concatenate them.
